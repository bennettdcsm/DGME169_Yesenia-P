Neurite Tracing Tool

A Python image analysis tool to quantify sensory nerve fibers in bladder fluorescence microscopy images.

Project Overview

During my time in the Beachy Lab at Stanford, I developed a custom Python tool to quantify CGRP-positive neurites in mouse bladder tissue. The goal was to support experiments investigating how estrogen affects bladder sensory innervation and urothelial homeostasis, which is relevant to chronic pelvic pain conditions such as Interstitial Cystitis.

The tool automated a process that was previously done manually, helping researchers analyze images faster, reduce human error, and compare nerve density across multiple samples in a consistent way.

Scientific Purpose

The Neurite Tracing Tool was designed to answer three main questions:

  • Does sensory innervation change with age or sex?
  • Does endogenous estrogen influence bladder nerve density?
  • Does supplemental estradiol affect sensory nerve density or urothelial organoid growth?

To support these experiments, the tool needed to reliably identify CGRP-labeled neurites, estimate their total length, and output a numeric value that could be compared between age groups, sexes, and hormone conditions.

How the Tool Works

1. Image Preprocessing

  • Convert fluorescence images to grayscale.
  • Apply Gaussian smoothing to reduce noise.
  • Adjust contrast to make neurites easier to detect.

2. Neurite Detection

  • Use thresholding to isolate bright CGRP-positive signal.
  • Apply morphological operations to clean up small artifacts.
  • Remove background noise and non-neurite regions.

3. Length Estimation

Once neurites are isolated, the tool estimates total neurite length by counting the number of “green” pixels and dividing by an estimated average neurite width:

Total neurite length ≈ green pixel count ÷ neurite width

This method matched the measurements used in the rest of the project and produced values that were consistent with manual neurite tracing by hand.

Research Findings Supported by the Tool

  • Sex and age differences: Female bladders tended to have higher sensory innervation than male bladders, and nerve density increased with age (P50 > P7).
  • Estrogen supplementation: Estradiol did not consistently increase sensory nerve density; in some cases high levels of estrogen appeared to impair nerve growth.
  • Organoid experiments: In separate assays, estrogen dose affected the size and number of urothelial organoids, which were quantified alongside neurite measurements.

The neurite length measurements from this tool were used directly in plots comparing age, sex, and hormone conditions across the study.

Tools Used

  • Python
  • OpenCV
  • NumPy / SciPy
  • Image Processing

Outcomes

This project resulted in a working, lab-tested tool for automated neurite quantification. It allowed the team to process many more images than would have been feasible by hand and became the foundation for my later, more advanced Fine-Tune Cell Annotation Tool.

Future Improvements

  • More accurate estimation of neurite width and branching.
  • Automatic reconnection of broken neurite segments.
  • Support for 3D or z-stack fluorescence images.
  • A simple UI so that non-programmers in the lab can run the tool without editing code.
