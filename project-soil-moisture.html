ESP32 Soil Moisture System

A smart plant monitor that sends live moisture and battery data to a custom SwiftUI iOS app using BLE.

Project Description

This project uses an ESP32 microcontroller to measure soil moisture and battery voltage, then sends the data over Bluetooth Low Energy to a custom iOS app built in SwiftUI. The system provides real-time plant monitoring for users.

Goals

  • Create an embedded system that can sense moisture levels accurately.
  • Send data wirelessly to a mobile device using BLE.
  • Display moisture + battery status in a user-friendly mobile UI.

Tools Used

  • ESP32
  • BLE
  • C / Arduino Framework
  • SwiftUI
  • iOS Development

Outcomes

The device successfully reads and transmits moisture and battery data, and the companion iPhone app displays the values and triggers notifications when dry conditions persist.
