ESP32 Soil Moisture System

A smart plant monitor that sends live moisture and battery data to a custom SwiftUI iOS app using BLE.

Project Description

This project uses an ESP32 microcontroller to measure soil moisture and battery voltage, then transmits the data over Bluetooth Low Energy (BLE) to a custom SwiftUI iOS application. The goal was to build a reliable plant-monitoring system that shows real-time moisture status and battery percentage in an easy-to-read mobile interface.

The device reads moisture levels through an analog sensor, monitors battery voltage using a voltage divider, and broadcasts both values over BLE. The companion iPhone app automatically receives updates, displays live readings, and sends alerts when the soil stays dry for extended periods.

Goals

  • Create an embedded system that can sense moisture levels accurately.
  • Send data wirelessly to a mobile device using BLE.
  • Display moisture + battery status in a user-friendly mobile UI.

Process

  1. Sensor reading and calibration. I connected a capacitive moisture sensor to the ESP32 and calibrated the raw analog values to match real soil conditions.
  2. Battery monitoring. I built a voltage divider circuit to safely measure Li-ion battery voltage using the ESP32 ADC, then converted the voltage to an estimated battery percentage.
  3. BLE service design. I created custom BLE characteristics for moisture and battery level so the iOS app could subscribe and receive live updates.
  4. SwiftUI app development. I built a simple app interface showing moisture status, battery percentage, and a live “dry soil” alert when low moisture readings persist.
  5. Full system testing. I tested the device with real plants, adjusted moisture thresholds, and verified that notifications triggered at the correct times.

Tools Used

  • ESP32
  • BLE
  • C / Arduino Framework
  • SwiftUI
  • iOS Development

Outcomes

The final device successfully measures and transmits both moisture and battery data in real time. The iOS app displays the readings clearly and sends notifications when the soil remains dry, helping users prevent underwatering. Adding BLE to the system made the device wireless and easy to monitor from anywhere in the home.

This project strengthened my skills in embedded systems, analog sensor calibration, BLE communication, and mobile app development. It also laid the foundation for integrating more sensors and expanding the system into a full plant-health monitoring network.
