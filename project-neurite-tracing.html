Neurite Tracing Tool

A Python tool made to trace and measure neurites in microscopy images.

Project Description

An early research tool developed for tracing neurites in microscopy images. The tool allowed researchers to measure lengths and shapes to support neuroscience experiments. This was my early introduction to image processing before later creating the more advanced Fine-Tune tool.

Tools Used

  • Python
  • OpenCV
  • Image Processing

Outcomes

This project helped me develop the foundation for my Fine-Tune annotation tool.
