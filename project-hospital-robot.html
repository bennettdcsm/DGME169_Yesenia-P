Hospital Sanitation Robot

Autonomous robot using multiple ESP32 microcontrollers, thermal sensing, and ultrasonic sensors to support sanitation workflows.

Project Description

This project expands on real challenges in hospital sanitation: staff shortages, inconsistent cleaning between patients, and the need for more frequent surface sanitization. The Autonomous Sanitation Assistant Robot (ASAR) was designed to help hospital housekeepers clean surfaces more efficiently, consistently, and with quantifiable verification.

ASAR navigates hospital environments, cleans target surfaces using a mounted disinfecting arm, and then uses thermal imaging combined with machine-learning inspired analysis to assess whether the cleaning was successful. The system integrates multi-ESP32 communication, multiple sensing modalities, and a mobile application for user control and cleaning validation.

Goals

  • Design a multi-ESP32 architecture where each microcontroller manages its own subsystem (arm motion, navigation, sensing).
  • Provide a reliable sanitizing routine controllable through a mobile app.
  • Clean surfaces more frequently and efficiently than standard manual workflows.
  • Include a verification step that uses thermal imaging to detect reduced bacterial activity after cleaning.

Process

  1. Identified real sanitation workflow problems in hospitals based on research and user interviews.
  2. Developed a multi-node ESP32 system with a central master board communicating via a TCA9548A I2C multiplexer.
  3. Built the mechanical arm with multi-segment motion for wiping surfaces using a predefined cleaning pattern.
  4. Integrated ultrasonic sensors, wheel modules, and H-Bridge motor drivers for autonomous movement.
  5. Added an ESP32-C3 camera and MLX90640 thermal array to measure surface temperature differences before and after cleaning.
  6. Designed a companion mobile app allowing users to log in, select devices, start cleaning tasks, and view cleanliness diagnostics.

How ASAR Determines Cleanliness

Research shows that metabolic heat from bacteria can be detected before visible growth appears. The MLX90640 infrared sensor captures a 768-pixel temperature array, and the system checks for a noticeable temperature decrease after cleaning as a sign that bacterial activity has been reduced. This gives hospital staff quantifiable validation instead of guesswork.

Tools Used

  • ESP32
  • ESP32-C3 (camera)
  • FreeRTOS
  • Ultrasonic Sensors
  • H-Bridge Motor Drivers
  • MLX90640 Thermal Camera
  • TCA9548A I2C Multiplexer
  • Mobile App (control & diagnostics)

Outcomes

ASAR successfully navigates controlled test spaces, detects high-priority zones using thermal imaging, cleans surfaces using its multi-segment arm, and evaluates cleaning effectiveness. The modular design allows future expansions, such as additional sensing modes or improved autonomy.

The final prototype demonstrates that low-cost, multi-microcontroller robots can support overworked hospital staff and improve overall sanitation quality.
