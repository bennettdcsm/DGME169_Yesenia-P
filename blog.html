Building a Multi-ESP32 Robotics Platform

Project Process • Robotics

This post breaks down how a complex robotics project was split into smaller embedded tasks, each handled by its own ESP32. From wiring the TCA9548A I2C switch to validating each state in the controller, the robot highlights practical strategies for debugging under real-world constraints.

Connecting an ESP32 to an iOS App with BLE

Project Notes • Connected Devices

A soil moisture monitor was built to send live sensor and battery data to a SwiftUI application. The write-up focuses on defining BLE services and characteristics, configuring notifications, and designing a simple interface that clearly communicates plant health to the user.

Designing the Fine-Tune Cell Annotation Workflow

Research Tools • Computer Vision

Fine-Tune is a Python-based annotation tool that overlays model-generated predictions on neuroscience images and allows quick adjustments with custom bounding boxes. This post describes the workflow behind the tool, from loading tiled images to updating predictions in real time, with a focus on making cell review faster and more accurate for researchers.
